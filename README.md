# Online Shopping Cart System

## 📌 Project Description
The Online Shopping Cart System is a web-based application developed to simplify online shopping. It allows users to browse products, add items to the cart, manage quantities, and automatically calculate the total bill, providing a smooth and user-friendly shopping experience.

---

## 🚀 Features
- 🛍️ Browse and select products  
- ➕ Add items to cart  
- 🔢 Manage product quantity  
- 💰 Automatic total bill calculation  
- 👤 User registration and login system  
- 📝 Profile management (view & update)  
- 🛒 View cart items  
- ❌ Remove items from cart  
- ⚡ Fast and user-friendly interface  

---

## 🛠️ Technologies Used
- 💻 C#  
- ⚙️ .NET Framework  
- 🧰 Microsoft Visual Studio  
- 🗄️ SQL Server 

---

## ▶️ How to Run the Project
1. Open the project in Visual Studio  
2. Restore NuGet packages  
3. Build the solution  
4. Run the application  
5. Access it via browser or local server
